Overview

To be able to classify topics accurately you need to understand language like a human. Using a deep-learning mathematical framework each text snippet can be classified into a topic with > 93% accuracy. This framework comprises the latest advancements in natural language processing and understands the contextual meaning of the individual sentence in relation to its adjacent sentences and its placement within the entire paragraph/document. By developing an AI that can understand the semantic and contextual nature of an individual sentence within its surrounding environment, the topic classification can provide a deeper level of understanding as meticulously as a human at scale.

Key Features

Key Features

  • Language has multi-scaled representation -

    It is looking at words that are near and far away from each other in terms of their relationships.

  • Builds memory across the board -

    The model has been trained to extract context from sentences, near and far, in the document to build memory in the system.

  • List of topics is expandable -

    Users can start adding their own list of topics which can better increase the accuracy and value of the AI.

  • Dynamic Learning -

    The model keeps growing in size, as well as gets smarter as it encounters new data and this is able to deliver more value over time.

  • Human-like understanding -

    The model achieves 93% accuracy with respect to the Bayes Error.

  • Rapid contextual learning -

    With a very limited training dataset the AI is able to rapidly learn the context

How Our Model Works

The model starts by converting the text into a multi-scale numerical representation. This representation allows the model to build a multi-scale memory from the text, and thus is able to extract the most relevant information related to the task. These numerical representations are then passed on to a custom architecture based deep-learning network module. We have been conducting extensive research to build a continuous dynamic learning network, that can learn from the feedback given by a user, without going into catastrophic forgetting, as is usually the case with online learning for deep-learning networks. Our network has the ability to semi-autonomously grow in size, to alleviate this issue as well as become smarter as it looks at more data, just like a child becomes an educated adult.
